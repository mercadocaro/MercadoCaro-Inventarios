# MercadoCaro Inventarios

Sistema de inventario desarrollado en Python para la gestión de refacciones de teléfonos celulares.

## Características

- Agregar producto individual
- Agregar teléfono completo
- Buscar productos por marca, modelo o pieza
- Visualizar inventario completo
- Guardado automático en CSV

## Tecnologías utilizadas

- Python
- CSV
- Visual Studio Code

## Estructura

- main.py
- main_v1.0.py

v1.0

Primera version funcional del sistema de inventario MercadoCaro.
